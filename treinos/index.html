1
Instalando as ferramentas
⏱ ~5 min
1.1
Abrir o Terminal
▼

No seu Mac, aperta Cmd + Espaço, digita Terminal e aperta Enter.

Vai abrir uma janela preta ou cinza. Não tem segredo — você vai colar uns comandos aqui.

1.2
Instalar o Homebrew
▼

Cola esse comando no Terminal e aperta Enter:

/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

O que vai acontecer:

  • Vai pedir sua senha do Mac (a mesma do login)
  • Digita e aperta Enter — não aparece nada na tela enquanto digita, é normal!
  • Espera terminar (3–5 minutos)
✅Quando terminar, vai aparecer uma mensagem com Next steps:
1.3
Adicionar Homebrew ao sistema
▼

Cola esses dois comandos, um de cada vez:

echo 'eval "$(/opt/homebrew/bin/brew shellenv)"' >> ~/.zprofile
source ~/.zprofile

Testa se funcionou:

brew --version
✅Se aparecer Homebrew 4.x.x, tá certo!
1.4
Instalar o Python 3.11
▼
brew install python@3.11
☕Pode demorar uns 5 minutos. Aproveita pra fazer um café!
2
Instalando o conector Garmin
⏱ ~5 min
2.1
Baixar o programa
▼

Cola esses comandos um de cada vez:

mkdir -p ~/garmin_mcp_server
cd ~/garmin_mcp_server
git clone https://github.com/Taxuspt/garmin_mcp.git .
2.2
Criar ambiente Python isolado
▼
/opt/homebrew/bin/python3.11 -m venv venv
source venv/bin/activate
ℹ️O terminal vai começar com (venv) — isso é normal e tá tudo certo.
2.3
Instalar dependências
▼
pip install --upgrade pip
pip install -e .
✅Quando aparecer Successfully installed garmin-mcp…, tá pronto!
3
Conectando com sua conta Garmin
⏱ ~2 min
3.1
Autenticar no Garmin Connect
▼
garmin-mcp-auth

O que vai acontecer:

  1. Vai pedir seu email do Garmin Connect
  2. Vai pedir sua senha do Garmin
  3. Se você tem 2 fatores (MFA) ativado, vai pedir o código
✅
Sucesso quando aparecer:
Authentication successful! · SUCCESS: You can now use the Garmin MCP server!
🎉Os tokens de acesso ficam salvos no seu Mac e duram 6 meses.
4
Configurando o Claude Desktop
⏱ ~3 min
4.1
Descobrir seu nome de usuário
▼
whoami

Vai aparecer seu nome de usuário do Mac (ex: joaopaulo). Anota — você vai usar agora.

4.2
Abrir o arquivo de configuração
▼
open ~/Library/Application\ Support/Claude/claude_desktop_config.json

Vai abrir um editor de texto com um arquivo JSON.

4.3
Substituir o conteúdo
▼

Apaga tudo que tá no arquivo e cola exatamente isso:

{ "mcpServers": { "garmin-local": { "command": "/Users/SEUNOME/garmin_mcp_server/venv/bin/garmin-mcp", "env": { "GARMIN_COOKIES_FILE": "/Users/SEUNOME/.garminconnect/cookies.json" } } } }
⚠️Substitua SEUNOME (nos 2 lugares) pelo nome que apareceu no whoami.
4.4
Salvar e reiniciar o Claude
▼
  1. Aperta Cmd + S para salvar o arquivo
  2. Feche o Claude Desktop completamente (Cmd + Q)
  3. Abra o Claude de novo
✅O Claude Desktop agora tem acesso ao servidor Garmin!

⏱ ~1 min

Solução de problemas
"Server disconnected" no Claude
▼

1. Verifica se você substituiu SEUNOME pelo seu usuário nos dois lugares no config.

2. Confirma que rodou o garmin-mcp-auth com sucesso (Parte 3).

3. No Claude Desktop, vai em Configurações → Servidores MCP locais → Ver Logs e lê a mensagem de erro.

"Command not found: brew"
▼

Você pulou o Passo 1.3. Roda esses dois comandos:

echo 'eval "$(/opt/homebrew/bin/brew shellenv)"' >> ~/.zprofile
source ~/.zprofile
"Module not found: garmin_mcp"
▼

O ambiente Python não foi ativado. Roda no Terminal:

cd ~/garmin_mcp_server source venv/bin/activate pip install -e .
Tokens expiraram (após ~6 meses)
▼
cd ~/garmin_mcp_server source venv/bin/activate garmin-mcp-auth
Outro erro? Dica de ouro
▼

O Claude pode te ajudar a resolver os próprios erros de instalação. Abre uma nova conversa e cola:

Tô seguindo um tutorial pra conectar meu Garmin ao Claude Desktop via MCP no Mac. Tô travado no passo X e aparece esse erro: [COLA O ERRO AQUI]. Me ajuda passo a passo.

Se o erro for em algum log, vai em Configurações → Servidores MCP locais → Ver Logs, copia tudo e cola na conversa.

É seguro?
Roda localmente

Tudo fica no seu Mac. Nenhum servidor de terceiros tem acesso aos seus dados.

Tokens seguros

Suas credenciais ficam em ~/.garminconnect só no seu computador.

Dados direto da fonte

Os dados vêm direto do servidor oficial do Garmin Connect.

Como desconectar

Delete ~/garmin_mcp_server e ~/.garminconnect e remova o bloco do config.
